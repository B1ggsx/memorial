# Memorial — Loretta Milner

A plain-HTML memorial page for Loretta Milner, hosted on GitHub Pages.

## Live Site

**[https://b1ggsx.github.io/memorial/](https://b1ggsx.github.io/memorial/)**

## Enabling GitHub Pages

1. Go to **Settings → Pages** in this repository:  
   `https://github.com/B1ggsx/memorial/settings/pages`

2. Under **Build and deployment**, set:
   - **Source**: `Deploy from a branch`
   - **Branch**: `main`
   - **Folder**: `/ (root)`

3. Click **Save**.

GitHub will build and publish the site within a minute or two. The live URL will appear at the top of the Pages settings page once it is ready.
